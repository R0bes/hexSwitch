# HexSwitch

Hexagonal runtime switchboard for config-driven microservices.

## Status

This repository is currently in its foundational scaffold phase. The project structure, tooling, and CI/CD pipelines are being established. No runtime behavior is implemented yet.

## Project Structure

- `src/hexswitch/` - Python package (skeleton)
- `tests/` - Test suite (unit and integration tests)
- `docs/` - Project documentation
- `docker/` - Dockerfiles and related scripts
- `infra/` - Infrastructure and deployment configuration
- `bin/` - Helper scripts
